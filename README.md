# jcore
Core of HHP.net website written using java

## Installing
Add to your gradle file
´´´
   repositories {
        maven { url "https://jitpack.io" }
   }
   dependencies {
         compile 'com.github.hhpnet:hhpnet-jcore:master-SNAPSHOT'
   }
